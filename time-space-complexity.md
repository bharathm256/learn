# BIG O notation
Best -> Omega
Avg -> Beta
Worst -> BIG (O)

### Various Big (O) Notations:
* O(1)
* O(log n)
* O(n)
* O(n log n)
* O(n2)

### Drop Constraint
O(2n) = O(n) - constant 2 can be dropped

### Drop non dominants
O(n2 + n) = O(n2) - n value is less here and can be dropped

### O(n log n)
example: O(a + b) , O(a * b)
